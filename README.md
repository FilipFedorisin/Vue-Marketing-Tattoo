# poorich-tattoo

## Build & Setup

```bash
# install dependencies
$ npm install
$ npm install --only=dev

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
